# Project-89  

课设作业

一人小组懒得守git规范了，提交信息是瞎写的  

目录标准：  

Scripts/Data/：存放所有承载单位静态属性的纯 C# 数据类（如基础攻防、装甲标签、行动力）以及对应的 JSON 解析器代码。

Scripts/Core/：存放战斗演算黑盒（如战损抛骰子、战斗优势分计算）与后勤泛洪寻路算法。这部分代码应当是纯粹的数学运算，完全不依赖 Godot 场景树节点。

Scripts/Battlefield/：存放网格系统、ZOC（控制区）动态投射、交界线提取算法以及回合全局状态机。

Scripts/UI/：存放与玩家交互的热座部署面板、兵牌 UI 更新等视图层逻辑。
